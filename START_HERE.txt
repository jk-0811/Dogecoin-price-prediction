⚡ QUICK START (Windows) - 2 STEPS

📍 Step 1: Run Setup Script
   Open command prompt in this folder and type:
   
   setup.bat
   
   Wait for completion... (installs everything)


📍 Step 2: Start Server
   After setup completes, type:
   
   python manage.py runserver
   
   Then open: http://localhost:8000


🎯 THAT'S IT! 

You now have:
🌐 Dashboard at http://localhost:8000
📊 API at http://localhost:8000/api/predict/
📈 Charts and visualizations
🤖 ML model making predictions


💡 NEXT STEPS

📊 INPUT FIELDS EXPLAINED

When making a prediction, provide:

Open          → Current opening price (e.g., 0.15)
High          → Daily high price (e.g., 0.16)
Low           → Daily low price (e.g., 0.14)
Volume        → Trading volume (e.g., 1250000000)
MA5           → 5-day moving average (e.g., 0.152)
MA10          → 10-day moving average (e.g., 0.151)
Daily Return  → % change (e.g., 0.025 = 2.5%)
Volatility    → Price volatility (e.g., 0.015)
